Laminat Catalog — финальная версия

Архив пересобран заново (свежий файл).
Если проект у вас уже запущен локально — используйте его.

Функциональность:
- Главная = каталог
- Поиск по названию
- Фильтр по оттенку
- Полноэкранный просмотр фото
- Админка: добавить / найти / удалить цвет
- Контакты
- Подвал: ООО «СОЮЗТОРГ-М»

Запуск:
npm install
создать .env с DATABASE_URL="file:./dev.db"
npm run migrate
npm run seed
npm run dev
