# Laminat Catalog (Next.js + Prisma + SQLite)

Готовый адаптивный сайт-каталог ламината:
- Главная = каталог (поиск + фильтры)
- Без страницы товара: клик → фото на весь экран, повторный клик → закрыть
- Контакты
- Админка: добавление нового цвета (фото + характеристики), удаление

## Запуск локально

1) Установи Node.js 18+  
2) В папке проекта:

```bash
npm i
cp .env.example .env
```

3) Создай БД и заполни демо-данными:

```bash
npm run migrate
npm run seed
```

4) Запусти:

```bash
npm run dev
```

Открой: http://localhost:3000

## Админка

Открой: `/admin`  
Вставь `ADMIN_TOKEN` (из `.env`) — он нужен для добавления/удаления.

## Фото

- В админке можно загружать несколько фото; они сохраняются в `public/uploads`.
- Для продакшена лучше хранить в S3/Cloudinary, но локальная загрузка уже работает.
